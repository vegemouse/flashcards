Flashcards

Javascript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

An operator in a programming language is a symbol that tells the compiler or interpreter to perform specific mathematical, relational or logical operation and produce final result.

Variables

A data item that may take on more than one value during the runtime of a program.

Variable Naming Conventions

A set of rules for choosing the character sequence to be used for identifiers which denote variables. Generally camelCase for Javascript.

Functions

A self-contained software routine that performs a task. Functions can do a large amount of processing or as little as adding two numbers and deriving a result. Values are passed to the function, and values may be returned.

Methods

A procedure associated with a message and an object. An object is made up of data and behavior, which form the interface that an object presents to the outside world.

Arguments

A value that is passed between programs, subroutines or functions. Arguments are independent items, or variables, that contain data or codes. When an argument is used to customize a program for a user, it is typically called a "parameter."

Parameters

A special kind of variable, used in a subroutine to refer to one of the pieces of data provided as input to the subroutine.

Return

Upon completion of a routine or function, to go back to the point in the program that called the operation. When a function returns control, it may also return a result in the form of a value.

Chaining

A technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.

Strings

Any finite sequence of characters (i.e., letters, numerals, symbols and punctuation marks).

Booleans

A data type, having two values (usually denoted true and false), intended to represent the truth values of logic and Boolean algebra

Undefined

Undefined value is a condition where an expression does not have a correct value, although it is syntactically correct. An undefined value must not be confused with empty string, boolean "false" or other "empty" (but defined) values.

NaN

Standing for not a number, it is a numeric data type value representing an undefined or unrepresentable value, especially in floating-point calculations.

Escape

A keyboard key commonly used to exit a mode or routine, or cancel some function. Used by typing "\" before characters.

Alert

An alert box is often used if you want to make sure information comes through to the user.

Comments

Lines in code used to make comments rather than affecting actual code on page.

jQuery

jQuery is a concise and fast JavaScript library that can be used to simplify event handling, HTML document traversing, Ajax interactions and animation for speedy website development. jQuery simplifies the HTML's client-side scripting, thus simplifying Web 2.0 applications development.

Attributes

A specification that defines a property of an object, element, or file. It may also refer to or set the specific value for a given instance of such. For clarity, attributes should more correctly be considered metadata.
